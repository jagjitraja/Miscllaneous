      • How to do the horizontal centering for 'fixed' and 'absolute' positioned elements? E.g., the link at the bottom.
        • You need to calculate the correct position, using 'calc(), vw, vh', where vw is for viewport width and vh is for viewport height. 0vw ~ 100vw, and 0vh ~ 100vw.
        • E.g., the horizontal center with fixed positioning; Note that calc(50vw-55px) does not work. It should be calc(50vw - 55px). Spaces around '-' are necessary. 
          <p style='width:110px; position:fixed; top:25%; left:???(50vw - 55px); background-color:Grey; text-align:center; z-index:1; font-size:200%'>
    <a href='//cs.tru.ca'>cs.tru.ca
</p>
