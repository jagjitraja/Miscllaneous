3.12 How to send data to the server?

  1. How to send data to the server?
    • Using a form
      • method: post, get
      • action: the URL of the server script
      • target: window (or iframe) to display the web content coming back from the action server script
    • Input, checkbox, radio, textarea, select, ... within a form have names and values. These pairs of name and value will be submitted to the URL specified in form-action, when the input-submit within the form is clicked.
    • What if name is omitted?
    • How are the HTTP get and post methods different? Read all in HTML Forms.
    • Can an HTML document receive the user input?
    • Here is an example. 
      <form ???='get' ???='controller.php'>
    <input type='hidden' name='page' value='StartPage'>
    <input type='???' name='command' value='SignIn'>
    Username:<br>
    <input type='text' name='username' required><br>
    Password:<br>
    <input type='???' name='password' ???><br>
    <input type='???' value='Submit'>
</form>

// controller.php; Of course we will discuss about PHP very soon.
<?php
    echo $_GET['page'] . '<br>';  // It should be ???.
    echo $_GET['command'] . '<br>';  // It should be ???.
